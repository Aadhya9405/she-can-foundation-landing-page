# She Can Foundation Landing Page

A modern, responsive, and animated single-page NGO website built for the Frontend Development Internship Task.

## 🌟 Features

* Responsive Design
* Modern NGO UI
* Smooth Animations
* Interactive Buttons
* Scroll Reveal Effects
* Animated Statistics Counter
* Sticky Navbar
* Premium Pastel Theme

## 🎯 Purpose

This project represents the mission of **She Can Foundation** — empowering women, supporting children, and promoting girls’ education.

## 🛠 Tech Stack

* HTML5
* CSS3
* JavaScript
